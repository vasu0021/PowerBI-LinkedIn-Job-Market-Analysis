#  LinkedIn Job Posting Duration Analysis (Power BI)

##  Project Overview
This project analyzes **LinkedIn job posting data** to understand how the **time since a job was posted** affects the **number of applications received**.  
The analysis is performed using **Power BI**, focusing on posting durations such as hours, days, weeks, and months.

---

## Objective
- Analyze LinkedIn job posting durations  
- Study the relationship between posting age and application count  
- Identify optimal posting periods for maximum candidate engagement
